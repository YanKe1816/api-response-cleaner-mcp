Support

If you need help with the API Response Cleaner App, contact sidcraigau@gmail.com.

How to Reach Support

Email your question or issue to sidcraigau@gmail.com. Include the raw response you submitted, the target fields you requested, and the actual result you received.

Appropriate Use Guidelines

Submit only structured JSON-like objects intended for deterministic field filtering. Do not use this App to request summaries, recommendations, or generated values. This App only extracts requested fields and reports missing or removed fields.

Feedback and Issue Reports

If you encounter a problem, send feedback with a short description of the issue, the expected cleaned output, and the actual output. This helps reproduce and review the issue quickly.
